# Changelog

All notable changes to Python ExamDiff Pro will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-31

### Added
- Initial release of Python ExamDiff Pro
- Myers' diff algorithm implementation for accurate file comparison
- Modern GUI built with CustomTkinter
- Side-by-side file comparison with syntax highlighting
- Directory comparison with recursive tree view
- Three-way merge support with conflict resolution
- Syntax highlighting for 25+ programming languages
- HTML and PDF report generation
- Command-line interface for automation
- Plugin system for extensibility
- Windows Explorer context menu integration
- Dark and light theme support
- Comprehensive keyboard shortcuts
- Session save/load functionality
- Bookmark system for important differences
- Git integration for comparing commits and branches
- Large file support (up to 1GB with chunked processing)
- Binary file comparison with hex dump
- Multiple encoding support (UTF-8, UTF-16, ASCII, auto-detect)
- Smart ignore options (whitespace, case, blank lines, comments)
- Fuzzy matching for similar lines
- Moving block detection
- Minimap overview for quick navigation
- Undo/redo support
- Recent files tracking
- Configuration management with YAML

### Technical
- Type hints throughout the codebase
- Comprehensive error handling and logging
- Unit tests for core algorithms
- PEP 8 compliant code style
- Extensive documentation and docstrings
